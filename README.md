# Running 

1. Starting Docker compose and dependencies
    ```
    docker-compose up -d
    ```

2. Execute Application
    ```
    sbt run
    ```

3. Execute Testing
    ```
    sbt test
    ```

4. Execute Coverage Report
    ```
    sbt coverageReport
    ```
